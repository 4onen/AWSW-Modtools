AwSW Modding Tools
==================

This project is all about extending a visual novel called Angels with Scaly Wings, or AwSW for short.
To know more about the game, visit `the official website`_ for more information.

Contributing
------------

You can contribute! Writers and drawers are needed to produce high-quality mods.
Programmers are needed to ensure a stable and easy-to-use modding framework.

If you need assistance, please make a Github account and `create an issue`_.
Alternatively, create a thread `on the Steam Workshop forums`_ and we'll try and get back to you.

.. _the official website: http://angelswithscalywings.com/
.. _create an issue: https://help.github.com/articles/creating-an-issue/
.. _on the Steam Workshop forums: https://steamcommunity.com/workshop/discussions/18446744073709551615/?appid=571880
